== Description ==

Alluric Admin is a Wordpress plugin that gives a new look for its Administration panels, with an in-built dropdown menu feature. It is currently in beta but available for download. Many thanks to Dean Robinson for the backend and Ozh for the dropdown menu code.

== Installation ==

1. Upload the folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You're done!
